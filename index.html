Эта страница о тегах html

Теги


"Голова"сайта(head)

Предназначен для хранения элементов, цель которых — помочь браузеру в работе с данными


"Тело"сайта(body)

Представляет собой контент (содержимое) документа HTML

Параграф(p)

Определяет текстовый абзац


Пропуск строки(br)

Устанавливает переход на другую строку в том месте, где этот тег находится


Изаброжение(img)

Предназначен для отображения на веб-странице изображений в графическом формате GIF, JPEG или PNG


Заголовок(h1)

Представляет собой наиболее важный заголовок первого уровня, а тег "h6" служит для обозначения заголовка шестого уровня и является наименее значительным. По умолчанию, заголовок первого уровня отображается самым крупным шрифтом жирного начертания, заголовки последующего уровня по размеру меньше.


Ссылка(a)

Добавляет ссылку


Программная ссылка(link)

Устанавливает связь с внешним документом вроде файла со стилями или со шрифтами


Контейнер(div)

Объеденяет несколко строчек в контейнер
